# Vendor Language v1.0 (Alpha Release) 🚀

Welcome to **Vendor**, a revolutionary bilingual programming language that allows you to write code in plain **English** and **Tamil (தமிழ்)**! 

Vendor is designed to make programming incredibly simple, accessible, and logic-driven without the hurdle of complex syntax.

## ✨ Features
* **Bilingual Support:** Write code seamlessly in English or Tamil.
* **No Complex Syntax:** No semicolons, no curly braces. Just plain words.
* **Core Capabilities:** Variables, Math Operations, If/Else Conditions, and Loops.

## 🛠️ How to Use

1. Create a file with the `.ven` extension (e.g., `test.ven`).
2. Write your code using the Vendor syntax.
3. Run the engine using your terminal: `vendor test.ven` (or `python engine.py`)

## 📖 Syntax Examples / உதாரணங்கள்

### 1. Variables / மாறிகள்
* **English:** `set score to 100`
* **Tamil:** `பதிவு மதிப்பெண் = 100`

### 2. Math / கணிதம்
* **English:** `add a and b`
* **Tamil:** `a மற்றும் b ஐ கூட்டு`

### 3. Output / திரையில் காட்ட
* **English:** `show Welcome to Vendor!`
* **Tamil:** `காட்டு வெண்டர் மொழிக்கு நல்வரவு!`

### 4. Conditions / நிபந்தனைகள்
* **English:** `if score > 40 then show Student Passed`
* **Tamil:** `score > 40 என்றால் காட்டு மாணவர் தேர்ச்சி`

### 5. Loops / சுழற்சிகள்
* **English:** `repeat 5 times show Hello`
* **Tamil:** `5 முறை திரும்பு காட்டு வணக்கம்`
